About Shopping Tracker

Shopping Tracker is a web app where you can keep track of your expenses and see a graph of them.
Shopping Tracker is built with pure JavaScript from scratch, and the pie graph is made by Google.
This is a front-end application, so all user data is stored locally on the user's device.

You can decide to delete all saved data by clicking here.

This project is open-source and the code is on github
